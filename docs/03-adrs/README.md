For which backlog rows are actually blocking code today vs. only awaiting a formal write-up, see [../00-overview/implementation-status.md](../00-overview/implementation-status.md).

## Status definitions

- **Accepted** — binding; architecture must reflect it.
- **Proposed/Backlog** — identified as needed; not yet written or ratified; must not be treated as decided.
- **Superseded / Rejected** — moved to [../archive/](../archive/); not canonical.

> The backlog above was harvested from a now-archived pre-implementation review ([../archive/reviews/pre-implementation-adr-gate.md](../archive/reviews/pre-implementation-adr-gate.md)) and a freeze review ([../archive/reviews/architecture-freeze-review.md](../archive/reviews/architecture-freeze-review.md)), with one later addition — "VCS/PR integration & Apply targets" (now [ADR-0010](ADR-0010-vcs-pr-integration-and-apply-targets.md), Accepted) — surfaced by [RFC-0003](../01-rfcs/RFC-0003-interactive-assistant-and-multi-executor-pipelines.md) §4.1/§6, not those reviews. [RFC-0004](../01-rfcs/RFC-0004-multi-executor-router-and-publish-policy.md) proposed concrete shapes for the four rows Executor contract, Routing, Knowledge, and VCS/PR against a specific multi-vendor worked example; three of those (Executor contract, VCS/PR, Routing & policy) are now Accepted above, only Knowledge remains backlog. Those reviews are historical; this index is the canonical statement of what remains to be decided.
>
> **A governance process now exists** — [ADR-0000](ADR-0000-governance-and-ratification-process.md), Accepted 2026-07-16. Decisions are ratified by the project's sole maintainer editing an RFC/ADR's Status row (no enforced comment period while there is one maintainer; see ADR-0000 Decision 3 for the trigger that changes this). [ADR-0004](ADR-0004-reusable-act-template-format-and-evolution-policy.md) (Reusable-Act template format & evolution policy), [ADR-0009](ADR-0009-cli-and-output-contract.md) (CLI & output contract), and [ADR-0006](ADR-0006-routing-and-policy.md) (Routing & policy) were all drafted and accepted under this process on 2026-07-20. The remaining backlog rows above still each need their own ADR drafted before that ratification step is even possible.
